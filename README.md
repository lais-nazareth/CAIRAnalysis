# Técnicas de Content Aware Image Retargeting (CAIR)
Uma análise comparativa de métodos de redimencionamento de imagens


## Como utilizar os códigos:
1- Informe o caminho da imagem que você deseja redimensionar e as dimensões alvo no main.py e execute-o.

2- Escolha o método desejado para o redimensionamento.
3- Os resultados estão na pasta "output" que foi criada dentro da pasta do método escolhido (ex: se escolhi Seam Carving, os resultados estão em Seam Carving/output).
