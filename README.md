# Técnicas de Content Aware Image Retargeting (CAIR)
Uma análise comparativa de métodos de redimencionamento de imagens


## Como utilizar os códigos:
1- Adicione o nome do arquivo da imagem e o nome da pasta em que ele se encontre, no arquivo py relativo ao método que se deseja testar (nos campos destacados no código);

2- Verifique no diretório [Nome do método] se a pasta original do arquivo se encontra;

3- Dentro desta pasta está o resultado do processamento da imagem.
